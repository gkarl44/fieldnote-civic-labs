## Publish options

This folder is plain static HTML and CSS. It can be published with GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any basic web host.

For a fast setup:

1. Register `fieldnoteciviclabs.com`.
2. Email is configured through Porkbun:
   `gordon@fieldnoteciviclabs.com` is a hosted mailbox for sending and receiving,
   `gkarl44@gmail.com` is verified in Gmail to send as `gordon@fieldnoteciviclabs.com`,
   and `info@fieldnoteciviclabs.com` forwards to `gkarl44@gmail.com`.
3. Deploy this folder as the site root.
4. Point the domain DNS to the selected host.
5. Update any temporary deployment URL in registration forms once the custom domain is live.
